# Assist Map（アシストマップ）

シンプルで軽量なナビ補助用の地図アプリです。  
サイクリング、登山、アウトドア、位置確認などに利用できます。

## 🌍 公開ページ

[https://ereyotti9163-oss.github.io/assist-map/](https://ereyotti9163-oss.github.io/assist-map/)

## 主な機能

* **OpenStreetMap** による世界地図表示
* **住所・施設名検索** / **緯度経度検索**
* 地図クリックによる**ピン設置**と**ルート表示**
* ピン間の**距離計算**（合計距離表示）
* ピン削除機能（直前のピンを取り消し）
* **ノースアップ**（北固定）
* **ヘディングアップ**（コンパス連動で進行方向を上に表示
* アシストマップアプリＵＲⅬ用ＱＲボタン(画面右下)タップで本アプリのＵＲⅬをＱＲコード化して表示
* スマートフォン対応 / **PWAアプリ化**（ホーム画面に追加可能）

## 使用技術

* HTML / JavaScript
* Leaflet
* OpenStreetMap
* Geolocation API / DeviceOrientation API

## 使い方

### 1. 検索
検索ボックスに以下の情報を入力して移動できます。
* 住所・施設名
* 緯度経度（例: `35.681236, 139.767125`）

### 2. ルート作成
地図をクリックするとピンが追加されます。ピンを繋ぐラインが表示され、画面に合計距離が計算・表示されます。

### 3. ピン削除
「ピン1つ消す」ボタンを押すと、最後に追加したピンを削除できます。

### 4. 地図の向き
* **ノースアップ**: 地図の北が常に上になります。
* **ヘディングアップ**: スマホのコンパスを使い、自分が向いている方向を上に表示します。

## 利用シーン
* サイクリングや登山のルート確認
* アウトドアでのナビ補助
* 緯度経度による正確な位置確認
* 万が一の遭難時の位置特定
